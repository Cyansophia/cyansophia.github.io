# Lanjia's Personal Website

Hi, I'm Lanjia — a psychology student learning to code. This repository
is the source of my personal website, where I share articles, paintings,
and small web tools I build while learning.

**Live site:** https://YOUR-USERNAME.github.io

## How it's built

No frameworks, no build step. Just handwritten HTML, CSS, and a little
JavaScript. Every page here is something I wrote myself as part of
learning.

## Structure

- `index.html` — home page
- `about.html` — about me
- `articles/` — essays and notes
- `paintings/` — image gallery
- `tools/` — tiny web widgets
- `assets/` — shared CSS, JS, and images

## Editing locally

1. Open the project folder in VS Code.
2. Right-click `index.html` → "Open with Live Server".
3. Edit any file, save, and the browser refreshes automatically.

## Publishing updates

```
git add .
git commit -m "describe what you changed"
git push
```

The live site updates within a minute.
